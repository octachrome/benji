The script is written using JSON. A script is one or more animations. Each animation can be either the name of an After Effects animation: "sit_down"; or an array of animations, which will be played in sequence: ["sit_down", "sitting"]; or an object with object with an "anim" property set to any kind of animation, and some other properties that change its behaviour:

  • "repeat_random": 5: repeat the animation approximately 5 times, with some random variation
  • "likelihood": 0.4: play the animation 40% of the time, and skip it the other 60% of the time
  • "backdrop": "bedroom": change the backdrop image before playing the animation
  • "name": "get_up": name a section of the script so it can be referenced in future
Lastly, an animation can be an object with a single property, "choice", set to an array of animations, each of which has a likelihood of being chosen. Only one of the choices will be played: {"choice": ["likelihood": 0.2, "anim": "rare_option"}, {"anim": "common_option"}]}. Any choices without an explicit likelihood will have the remaining probability split between them.

Animations be nested and combined together, e.g.: 

[
    "first_in_sequence",
    "second_in_sequence",
    {
        "choice": [
            {
                "likelihood": 0.2,
                "anim": [
                    "rare_third_in_sequence_step1",
                    "rare_third_in_sequence_step2"
                ]
            },
            {
                "likelihood": 0.8,
                "anim": "common_third_in_sequence"
            }
        ]
    },
    {
        "repeat_random": 10,
        "anim": "fourth_repeated_approx_ten_times"
    }
]
